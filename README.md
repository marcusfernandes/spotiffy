## Clone de layout Spotify

Esse projeto é um clone de duas telas do Spofity, utilizando React Native e Styled Components.

### Como instalar e executar
- ``` git clone https://github.com/marcusfernandes/spotiffy/ ```
- Entre da pasta do projeto ```cd spotiffy ```
- ``` yarn install ``` (instalar todas as dependências)
- ``` npm start` `` (iniciar)
